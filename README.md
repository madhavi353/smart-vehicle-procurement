# Smart Vehicle Procurement Using Blockchain Technology

This project provides a secure and transparent vehicle procurement system using blockchain.

## Features
- User Registration & Login
- Vehicle Request System
- Admin Approval
- Blockchain Transaction Storage

## Technologies Used
- Python (Django)
- HTML, CSS, JavaScript
- Blockchain Concepts
- SQLite Database

## How to Run
1. Install dependencies
2. Run server:
   python manage.py runserver
3. Open browser: http://127.0.0.1:8000
